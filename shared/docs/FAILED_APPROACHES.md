# Failed approaches

A running log of approaches that were tried and rejected, with enough context for future agents (human or AI) to avoid re-treading the same ground.

## Format

```
## YYYY-MM-DD — Short title
**Context:** what problem prompted the attempt
**Approach:** what was tried
**Why it failed:** concrete reason (not "it didn't work")
**Alternative chosen:** what we did instead (link to ADR if one was written)
```
